[ SEATTLE UFO BATTLE - ABOUT ]

[ Game Concept ]

Welcome to Seattle UFO Battle! This is an experimental project exploring game development using AI assistance. Fly your plane through a stylized Seattle, defend the Space Needle from incoming UFOs, and see how long you can last!

Built primarily with AI assistance to explore and learn. Expect some quirks – it's part of the fun!

[ How to Play ]

  • W/S or ↑/↓ : Pitch Up/Down
  • A/D or ←/→ : Roll Left/Right
  • E : Accelerate
  • Q : Brake/Decelerate
  • SPACE : Shoot Laser
  • C : Change Camera View
  • P : Pause Game
